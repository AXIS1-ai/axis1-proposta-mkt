# AXIS 1 — Página de Proposta Premium

Sistema gratuito em HTML/CSS/JS para rodar no GitHub Pages.

## Arquivos
- `index.html`
- `style.css`
- `script.js`

## Como usar
1. Suba os arquivos em um repositório público no GitHub.
2. Ative o GitHub Pages em Settings > Pages.
3. Abra o link gerado.
4. Cole o link do seu gerador de contratos no campo "Link do gerador de contratos".
5. Clique em "Salvar link".
6. Escolha o plano e envie a proposta ao cliente.

## Integração com contrato
O botão "Quero contratar esse plano" abre o gerador de contratos com o plano selecionado no link:

`?plano=estrategico`

Exemplo:
`https://seuusuario.github.io/axis1-contratos/?plano=estrategico`

## Importante
Para o plano abrir automaticamente no contrato, o gerador precisa ler o parâmetro `plano` na URL. Caso ainda não esteja lendo, peça para o Léo ajustar a versão do contrato.
